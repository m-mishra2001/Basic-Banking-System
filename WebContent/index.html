Basic Banking System

The Sparks Foundation's Basic Banking System internship Project is a web application through which we can perform Various Banking related Activities

Following is the functionality of this web application :- We can add a new Customer,Can View the details of all Customers ,Search for a particular customer by giving account no. ,Delete a particular customer,Edit details/information of previous customers, perform money transaction between customers. We can also View all the previous transaction happened through Transaction History, with various transaction details like sender's account number,receiver's account number,transaction id,amount,date and time of transaction ,transaction history also have functinality to delete a perticular history .
